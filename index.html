Resumo do Projeto

Este projeto demonstra:

  • Cálculo da sequência de Fibonacci com Python;
  • Somatório até atingir valores específicos (ex.: 88);
  • Derivada espacial usando Transformada de Fourier;
  • Animação de ondas 1D;
  • Derivada inversa de sinais via integração numérica;
  • Soma de múltiplas ondas senoidais.
