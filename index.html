Capacidade no Cálculo para Python

Este artigo apresenta experimentações acadêmicas em Python, abordando cálculos numéricos, transformadas de Fourier, integração, derivadas e superposição de ondas, com visualizações gráficas publicadas no repositório do GitHub.

1. Sequência de Fibonacci

Geração dos 10 primeiros números e soma incremental até atingir 88. Código implementado em Python usando listas e loops.

Função: cálculo de sequência e soma acumulativa.

Ver plot do Fibonacci

2. Derivada Espacial via Fourier

Uso da FFT para derivadas espaciais de funções periódicas. Comparação entre derivada calculada via Fourier e derivada analítica.

Ver plot da derivada Fourier

3. Simulação de Onda 1D via Fourier

Propagação de um pulso gaussiano usando esquema de segunda ordem no tempo e derivadas via FFT. Animação criada com matplotlib.animation.FuncAnimation.

Ver plot da simulação de onda

4. Derivada Inversa (Integração Numérica)

Cálculo da integral acumulativa de um sinal f(t), equivalente à derivada inversa. Método simples de integração numérica usando numpy.cumsum.

Ver plot da derivada inversa

5. Superposição de Ondas Senoidais

Soma de múltiplas ondas com diferentes amplitudes, frequências e fases. Demonstra o princípio de superposição e visualiza a primeira onda versus a soma total.

O código permite ajustar parâmetros de tempo, amplitude e fase para experimentar diferentes sinais.
